Proceso Possion¶

Definición 1¶

Sean $T_1$, $T_2$, ..., $T_n$ v.a.i.i.d con distribuci√≥n exponencial de par√°metro $\lambda$, : $\mathcal{E}xp(\lambda)$. : $\mathcal{W}_0=0$, $\mathcal{W}_n=T_1, T_2, ..., T_n$ para $n\geq 1$. Definimos el proceso Poisson de par√°metro o intensidad $\lambda$ por

\begin{equation} N(t) = m√°x \{n\geq 1, \ \ \mathcal{W}_n = T_1 + T_2 + ...+ T_n \leq t\} \end{equation}

Las variables aleatorias $T_n$ representan los intervalos de tiempo entre eventos sucesivos, y $\mathcal{W}_n = T_1, T_2, ..., T_n$ representa el instante en el que ocurre el n-√©simo evento, y $N(t)$ es el n√∫mero de eventos que han ocurrido hasta el instante t.

Proposición:¶

La variable aleatoria $N(t)$ tiene distribuci√≥n Poisson con par√°metro $(\lambda t)$, es decir, para cualquier $t>0$, y para $n=0, 1, ...$

\begin{equation} P(N(t) = n) = e^{-\lambda t} \frac{(\lambda t)^n}{n!} \end{equation}

Su valor esperado, y varianza son

$$E(N(t)) = \lambda t $$$$Var(N(t)) = \lambda t $$

# Funci√≥n para simular una trajectoria del proceso Poisson homog√©neo
sim.one.PoissonProcess <- function(run, tmax, lambda){
  w <- c()
  w[1] <- 0
  i <- 2
  while(w[i-1] < tmax){
    #i <- i + 1
    Ti <- rexp(1, lambda)
    #print(Ti)
    if(w[i-1] +  Ti < tmax){
      w[i] <- w[i-1] +  Ti
    }else{
      break
    }
    i <- i + 1
  } 
  df <- data.frame('runs' = rep(run, length(w)),  
                   'n' = 0:(length(w)-1),
                   't' = w)
  return(df)
}

# Funci√≥n para simular n trajectorias del proceso Poisson homog√©neo
sim.PoissonProcess <- function(n.runs, tmax, lambda){
  for(i in 1:n.runs){
    if(i == 1){
      df_1 <- sim.one.PoissonProcess(run=i, tmax, lambda)
    }else{
      df_i <- sim.one.PoissonProcess(run=i, tmax, lambda)
      df_1 <- rbind(df_1, df_i)
    }
    
  }
  return(df_1)
}

$$E(N(t)) = \lambda t $$$$Var(N(t)) = \lambda t $$

Para este ejemplo con t=50 y lambda = 0.2:

$$E(N(t)) = 0.2(50) = 10$$$$Var(N(t)) = 0.2(50) = 10$$

Ccovarianza¶

Para un proceso Poisson con par√°metro lambda $\lambda$, y $s<t$ la covarianza entre $N(s)$ y $N(t)$ es

\begin{equation} Cov(N(s), N(t)) = \lambda s \end{equation}

Ejemplo:

Obtener:

1.- $Cov(N(5), N(6))$

2.- $Cov(N(10), N(100))$

Soluci√≥n te√≥rica:

1.- $Cov(N(5), N(6))= \lambda s = (0.2)(5)= 1$

2.- $Cov(N(100), N(40))= \lambda s = (0.2)(40)= 8$
